# Тестовое задание для кандидатов

## Цель

Проверить ваши навыки работы с мобильными приложениями, анализа конкурентов и базовой технической грамотности.

---

## Задание 1: Анализ мобильного приложения

**Что нужно сделать:**

1. Скачайте APK приложения по ссылке: `[ДОБАВЛЮ ССЫЛКУ]`
2. Установите на устройство и протестируйте
3. Запишите скринкаст видео со своего телефона () где покажете:
    - Основной функционал приложения
    - Найденные баги или странности
    - Ваше общее впечатление

**Формат:** Видео на англиском языке пример: `[ДОБАВЛЮ ПРИМЕР]`

**Где отправить:** Ссылку на видео (YouTube/облако) в форме

---

## Задание 2: Детальный анализ конкурента

**Заполните Google форму:** `[ДОБАВЛЮ ССЫЛКУ]`

Вопросы будут по конкретному приложению/сервису. Примеры вопросов:

- Какая кнопка находится слева в главном меню?
- Сколько стоит месячная подписка Premium?
- Какие способы оплаты доступны в приложении?
- Какой текст на главной кнопке экрана онбординга?

**Важно:** Ответы проверяемые. ЛЛМ не поможет — нужно открыть приложение и посмотреть самому.

---

## Задание 3: Работа с GitHub

1. Создайте или используйте существующий GitHub аккаунт
2. Зайдите в репозиторий: `[ДОБАВЛЮ ССЫЛКУ]`
3. Сделайте форк репозитория
4. Создайте файл с ответами на вопросы (в репо будет список вопросов)
5. Сделайте Pull Request с вашими ответами

**Проверяем:** Умеете ли работать с Git, виден ли профиль, базовые навыки.

---

## Задание 4: Техническое задание на сервис

**Выберите один из сервисов:** `[ДОБАВЛЮ СПИСОК]`

Напишите короткое ТЗ (1-2 страницы) которое включает:

1. **Описание сервиса** — своими словами, что он делает
2. **Основной функционал** — ключевые возможности
3. **Цены** — актуальные тарифы с сайта/приложения
4. **Целевая аудитория** — кто использует
5. **Конкуренты** — 2-3 похожих сервиса

**Формат:** Google Doc или PDF

---

## Задание 5: Ответы на вопросы фрилансеров

Представьте, что вы менеджер проекта. Ответьте на вопросы от условных фрилансеров:

**Список вопросов:** `[ДОБАВЛЮ В ФОРМУ]`

Пример вопросов:

- "Какой цвет использовать для кнопки?"
- "Не понял задачу про экран авторизации"
- "Можно я сделаю по-другому?"

**Проверяем:** Как вы коммуницируете, даете ли чёткие ответы, умеете ли направлять.

---

## Важные правила

⚠️ **Вопросы задавайте ТОЛЬКО если полностью заблокированы и не можете двигаться дальше**

Всё остальное решается через:

- Google поиск
- ЛЛМ (ChatGPT, Claude и т.д.)
- Самостоятельный анализ

Мы проверяем вашу самостоятельность и resourcefulness.

## Как отправить результаты

Заполните финальную форму: `[ДОБАВЛЮ ССЫЛКУ]`

**В форме укажите:**

- Ссылку на видео (Задание 1)
- Ссылку на заполненную форму анализа (Задание 2)
- Ссылку на ваш Pull Request (Задание 3)
- Ссылку на ТЗ (Задание 4)
- Ответы на вопросы фрилансеров (Задание 5)
